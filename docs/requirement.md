## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | Amazon アフィリエイトブログ |
| サイト種別 | ブログ形式のアフィリエイトサイト |
| ジャンル | 未定（複数ジャンル対応可能な設計） |
| 収益モデル | Amazonアソシエイト（アフィリエイトリンク経由の商品購入による報酬） |
| API連携 | Amazon Product Advertising API (PA-API 5.0) |

## 2. 機能要件

### 2.1 記事管理

- Markdown/MDX で記事を作成・管理
- フロントマター（YAML）でタイトル、カテゴリ、タグ、日付、説明文を定義
- 記事内にカスタムコンポーネント（商品カード等）を埋め込み可能
- 下書き（draft）機能で非公開記事を管理

### 2.2 Amazon 商品連携

- PA-API 5.0 で商品情報を自動取得（商品名、価格、画像、評価、URL）
- ASIN を指定して記事内に商品カードを表示
- 商品情報を JSON ファイルにキャッシュ（API レート制限対策 + ビルド高速化）
- キャッシュの有効期限管理（24時間）
- API 失敗時はキャッシュからフォールバック表示

### 2.3 商品表示コンポーネント

- **商品カード**: 画像・商品名・価格・評価・アフィリエイトリンクをカード形式で表示
- **商品比較テーブル**: 複数商品をテーブルで比較表示
- **ランキングリスト**: 順位付きで商品を一覧表示

### 2.4 SEO 対策

- 各ページのメタタグ（title, description, OGP）自動生成
- 構造化データ（JSON-LD）対応（Article, Product, BreadcrumbList）
- サイトマップ（sitemap.xml）自動生成
- RSS フィード生成
- パンくずリスト
- canonical URL の設定

### 2.5 サイト機能

- レスポンシブデザイン（モバイルファースト）
- カテゴリ別記事一覧ページ
- タグ別記事一覧ページ
- 検索機能（クライアントサイド全文検索）
- ページネーション
- ダークモード切替
- 関連記事の自動表示
- ソーシャルシェアボタン

## 6. 制約事項・前提条件

- Amazon アソシエイトプログラムへの登録が必要
- PA-API の利用には売上実績（過去30日以内に3件以上の適格販売）が必要
- PA-API のレート制限: 1秒あたり1リクエスト（初期）
- アフィリエイトリンクであることの明示が法的に必要（景品表示法・ステマ規制対応）
- Amazon ブランドガイドラインに準拠した商品情報の表示が必要
